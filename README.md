# 🌐 Learning Project: HTML & CSS

This repository contains a website I built while taking an HTML and CSS course.
The project was created to practice basic layout, page structure, and styling.

## 🚀 About the Project

* Layout based on a course design
* Built with pure HTML5 and CSS3
* Responsive elements
* Flexbox practice
* Semantic markup

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox

## 📂 Project Structure

```
project-folder/
│── index.html
│── css/
│   └── styles.css
│── images/
└── README.md
```

## 🎯 Project Goal

To reinforce knowledge gained while learning:

* Basic HTML structure
* Working with classes and selectors
* Element positioning
* Responsive layout
* Spacing and typography

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/alexfrxx/Web-Studio.git
```

2. Open the `index.html` file in your browser
